# Website Testing Suite

Автоматизированная система тестирования веб-сайта на основе `pytest` с генерацией красивых PDF и Markdown отчётов. 

Фокус проекта сделан на проверке различных аспектов качества сайта (SEO, a11y, HTTP-статусы, производительность) с максимально простой и понятной системой построения документации для тестировщиков, инженеров, стейкхолдеров и HR-специалистов.

## Особенности
- Разделение тестов по категориям: SEO, производительность, доступность (a11y), функционал, поиск битых ссылок.
- Автоматическая генерация отчетов (`reports/REPORT.pdf`, `reports/REPORT.md`, `reports/BUGS.md`).
- Формирование PDF происходит "под капотом" с использованием Playwright и строковой шаблонизации Jinja2, без необходимости локально поднимать веб-сервисы.
- Динамический расчет _Quality Score_ на базе прохождения тестов.

## Установка

1. Создайте виртуальное окружение:  
   ```bash
   python -m venv venv
   ```
2. Активируйте виртуальное окружение:  
   - Mac/Linux: `source venv/bin/activate`
3. Установите зависимости:  
   ```bash
   pip install -r requirements.txt
   ```
4. Установите браузер Chromium (для генерации PDF):  
   ```bash
   playwright install chromium
   ```

## Запуск

Выполните скрипт, который запустит все тесты `pytest` и самостоятельно соберет отчеты:
```bash
python run_tests.py
```

После завершения тестов все результаты будут доступны в `reports/REPORT.pdf`.
